# Code for *Exploration of Fluxonium Parameters for Capacitive Cross-Resonance Gates*

This repository contains the Jupyter notebooks used for calculating
(in `python/notebooks/Calculations/`)
and plotting
(in `python/notebooks/Plotting/`)
simulation data
for the paper
*Exploration of Fluxonium Parameters for Capacitive Cross-Resonance Gates*.
The notebooks depend on the `fluxoniumcr` package,
which can be installed by running the following at the root of the repository:
```bash
pip install --editable ./python
```
This package must be installed as editable for the notebooks to correctly locate the `data/` directory.

## Data repository

The calculation notebooks save the results in the `data/` directory, but these are not included in this repository.
Instead, they can be downloaded
[from a separate 4TU.ResearchData dataset](example.org)
as a single `data.zip` file,
and then extracted into the `data/` directory so that the plotting notebooks can find them.

All numerical computations were performed on a mid-range laptop with an Intel Core i7-1260P CPU and 16 GB of RAM.
Depending on the calculation notebook,
the execution time can vary from a few minutes to over a day.
